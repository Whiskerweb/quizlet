# Logo CARDZ

Placez votre fichier logo ici avec le nom `logo.png` (ou `logo.svg`, `logo.jpg` selon le format).

Le logo sera automatiquement utilisé dans :
- Le header de la page d'accueil
- La navbar
- Le footer

**Format recommandé :** PNG avec fond transparent ou SVG
**Taille recommandée :** Au moins 80x80px pour une bonne qualité sur tous les écrans
